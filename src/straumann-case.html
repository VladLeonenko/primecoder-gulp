О проекте

Унаследовав секреты легендарного швейцарского инженерного искусства, Straumann Group объединяет в себе страсть к инновациям и бескомпромиссному качеству, предлагая специалистам и пациентам по всему миру передовые стоматологические решения.

Задачи

Разработать новый дизайн для сайта каталога, внедрить его в существующию систему, разработка новых модулей для оптимизации работы между отделами.

Решение

Разработка UI/UX макета и веб-сайта для оптово-розничной торговли сайта каталога straumann. Устранения ошибок и внедрение новых файлов на сервер Заказчика. Сохраненение позиций в поисковой выдаче. Разработан уникальный модуль для оптимизации работы между отделами.
